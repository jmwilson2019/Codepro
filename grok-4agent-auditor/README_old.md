Grok 4-Agent Auditor\n\nEnterprise-level code auditing powered by 4 collaborating Grok AI agents (you provide API keys).\n\n## Quick Start\n\n1. Open terminal in grok-4agent-auditor folder (or open that folder as workspace root).\n2. Run npm run watch to compile in watch mode.\n3. Press F5 to launch Extension Development Host window.\n\nIn the extension host window:\n- Go to Extensions view, confirm Grok 4-Agent Auditor is loaded.\n- Open Settings (Ctrl+,), search grok4agent, add your 4 Grok API keys to the array.\n- Open any code file, run command Grok 4-Agent: Audit Current File via Ctrl+Shift+P.\n\nThe extension will:\n- Call your 4 API keys in parallel (Architect, Security, Performance, Quality agents).\n- Synthesize a final executive report.\n- Open the Markdown report in a new editor tab.\n\n## Models\n- Default: grok-4.20-0309-reasoning\n- Change in settings: grok4agent.defaultModel\n\n## Next Features\nSay "add project audit", "add webview panel", etc. to expand.\n\n## Package for Distribution\nnpm run package → creates .vsix file. #why can't we just hit a button or a keyboard short cut, we shouldn't need to go to development
